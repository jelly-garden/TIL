# 문서 구조 관련 선택자

## 자식 선택자 - '>'
- 부모와 자식 관계
    ```css
    div > h1 { color: red; }
    ```

## 자손 선택자 - '공백'
- 조상과 자손 관계
    ```css
    div span { color: red; }
    ```

## 인접 형제 선택자 - '+'
- 형제 관계이면서 바로 뒤에 인접해 있는 요소
    ```css
    div + p { color: red; }
    ```

## 형제 선택자 - '~'
- 형제 관계이면서 뒤에 인접해 있는 요소
    ```css
    div ~ p { color: red; }
    ```
